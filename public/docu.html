API Documentation

Create a QR Code

Endpoint /qr
Method POST
Request Body 
for generate qr from Text
{
  "type": "Text",
  "content": "Hello, World!"
}

for generate qr for Number
{
    "type": "SMS",
    "content": "0987654321"
}

for generate qr for your gmail service
{
    "type": "Email",
    "content": "xyz@gmail.com"
}

for generate qr for your website,Social Media and Urls
{
    "type": "Url",
    "content": "www.google.in"
}

for generate qr for your protected wifi
{
    "type": "Wifi",
    "content": {
        "ssid":"Tp_link",
        "password":"7y23jge7",
        "isHidden":false
    }
}
